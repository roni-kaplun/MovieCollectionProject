# Style Guide

## Colours
These colours should be used across the application to keep the look consistent.

- Dark Blue: #0B1320  
- Dark Gray: #1E293B  
- Blue (Accent): #3B82F6  
- Green (Success): #22C55E  
- Orange (Warning): #F59E0B  

## Fonts
The application uses simple system fonts for readability.

- Font family: system-ui, Arial, sans-serif
- Main heading (h1): 2rem
- Sub heading (h2): 1.5rem
- Normal text: 1rem

## General Rules
- Use the blue accent colour for buttons or highlights only.
- Keep text easy to read with good contrast.
- Try to keep the design simple and clean.
