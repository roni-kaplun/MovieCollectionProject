## General Description

As a movie lover, I want to be able to create a list of movies I have watched for recording purposes.

As a user, I want to see movie information such as cast, movie length, genre, and the cover so that I can learn more about movies I plan to watch.

As a critic, I want to able to write reviews for movies and compare my 5 star rating to the general consensus.
